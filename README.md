# Arcus

**A R C U S** — A personal AI assistant interface.

Arcus is a sleek, voice-ready assistant UI inspired by JARVIS. This repo focuses on the front-end shell; backend and AI integration are handled separately.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Stack

- React 19 + TypeScript
- Vite
- CSS (no extra UI framework — kept lightweight)

## Project structure

```
Arcus/
```
